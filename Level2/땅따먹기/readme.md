# 프로그래머스 연습문제에서 나온 level2 문제입니다.
## 이 문제를 스스로 풀지 못해 "나의풀이"는 없습니다.
## "다른사람풀이"에는 다른 사람이 문제를 푼 코드와 그것에 해석하며 공부한 내용을 적었습니다.
## "다른사람풀이(동적계획법 - 오답)"에는 다른 사람이 동적계획법으로 문제를 푼 코드(오답)와 그것에 해석하며 공부한 내용을 적었습니다.
